Travlr Getaways Travel Booking Website

About the Project

The Travlr Getaways Travel Booking Website is a comprehensive web application developed as part of the CS465 Full Stack Development course. It allows users to browse and book travel packages, featuring user authentication and trip management functionalities.

Enhancements

In this project, I refactored the code to implement a modular, component-based architecture using Angular. This enhancement improved the maintainability and scalability of the application.

Skills Demonstrated

  • Front-end Development with Angular
  • Modular Software Design
  • Component-Based Architecture
  • Dependency Injection and Service Management
